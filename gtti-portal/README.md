## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn
- Anthropic API Key → [Get one here](https://console.anthropic.com)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/gtti-smart-portal.git
cd gtti-smart-portal

# 2. Install dependencies
npm install

# 3. Set up environment variables
cp .env.example .env
# Edit .env and add your ANTHROPIC_API_KEY

# 4. Set up the database
npx prisma generate
npx prisma db push

# 5. Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser! 🎉

## ⚙️ Environment Variables

Create a `.env` file in the root directory:

```env
# Database (SQLite for development)
DATABASE_URL="file:./dev.db"

# NextAuth
NEXTAUTH_SECRET="your-random-secret-key"
NEXTAUTH_URL="http://localhost:3000"

# Anthropic Claude AI ← REQUIRED for AI features
ANTHROPIC_API_KEY="sk-ant-your-key-here"
```

## 🤖 AI Features (Powered by Claude)

### 1. AI CV Builder
- Generates ATS-optimized, professional CVs
- Enhances career objectives
- Adds industry-specific keywords

### 2. AI Student Chatbot

- Career guidance for technical graduates
- Course and scholarship information
- Interview preparation tips

### 3. AI Lesson Simplifier
- Breaks down complex technical topics
- Uses local Pakistani examples/analogies
- Supports Beginner/Intermediate/Advanced levels
- Generates practice questions

## 🗄️ Database

Uses **Prisma ORM** with **SQLite** (development) or **PostgreSQL** (production).

```bash
# View database in browser
npx prisma studio

# Reset database
npx prisma db push --force-reset

# Generate Prisma client after schema changes
npx prisma generate
```

## 🚢 Deployment

### Deploy to Vercel (Recommended)

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel

# Set environment variables in Vercel Dashboard
# Add: ANTHROPIC_API_KEY, NEXTAUTH_SECRET, DATABASE_URL
```

### Deploy to other platforms
The app is a standard Next.js application and can be deployed to:
- **Railway** (with PostgreSQL)
- **Render**
- **DigitalOcean App Platform**
